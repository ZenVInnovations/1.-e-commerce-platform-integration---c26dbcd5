# 💰 E-Commerce Platform integration

## 📌 Objective
To build a dynamic, scalable e-commerce platform with a user-friendly interface, leveraging full-stack JavaScript for efficient development, robust data management, and seamless integration of essential e-commerce functionalities.

---

## 🧭 Project Flow

1. ✅ **Phase 1: Research & Documentation**  
2. 📊 **Phase 2: Design** – Create system flowchart using draw.io  
3. 👨‍💻 **Phase 3: Development** – Code models and preprocessing  
4. 🧪 **Phase 4: Testing** – Evaluate models with various inputs  
5. 🚀 **Phase 5: Deployment** – Build UI in Streamlit and deploy to VS Code

## 🧠 Models Used

- **User** - Represents customers and possibly admin users.  
- **Product** – Represents items for sale.
- **Order** -Tracks user purchasas.
- **Payment** -For integrating payment gateways.
